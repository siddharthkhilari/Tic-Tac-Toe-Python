# Tic Tac Toe Game (Python)

This is a console-based Tic Tac Toe game built using Python.

In this game, the user plays as **Player X** and the computer plays as **Player O**.
The computer selects moves randomly from available positions.

---

## Features

- Human vs Computer gameplay
- 3x3 Tic Tac Toe board
- Input validation (prevents wrong or repeated positions)
- Automatic computer move
- Winner detection
- Draw condition handling

---

## Technologies Used

- Python
